NoReact – Allergen Detection App

The NoReact mobile application was developed for the purpose of assisting users in the detection of allergens in food products. This objective is pursued by means of intelligent image recognition and real-time AI assistance. The application was built using Android Studio (Java), integrated with Roboflow for image classification, Firebase for authentication and database, and Gemini API for AI chat functionality.

Year: 2025
Role: Backend Developer
Course: Mobile Programming
Repository: https://github.com/Ikannilaaa/NoReact.git
Documentation: https://youtube.com/shorts/TCgr6HdFFc4

Project Description

NoReact is a smart mobile application designed to assist users, especially those with food allergies, in identifying potential allergens in their meals. The application provides comprehensive features to ensure food safety and user convenience.

Key Features

  • Home Screen: Users can explore a list of common allergens and browse foods categorized as allergen-free (e.g., egg-free). Each food item is also linked to a recipe for user reference.
  • AI Chat: Integrated with Gemini API, this feature allows users to interact with an AI assistant for allergy-related questions, food suggestions, or recipe customization.
  • Food Scanner: Users can scan food by capturing an image via camera or selecting one from the gallery. The app then predicts the name of the food and lists any potential allergens based on trained models from Roboflow.
  • History: Stores a log of previously scanned foods, including prediction results and allergen detection, allowing users to track their scans over time.
Java Android Studio Roboflow Firebase Gemini API

GAYOCRAFT - Game Visual Novel

Created an educational simulation game called GAYOCRAFT, developed using Ren'Py, which takes players on a journey to become a successful Gayo coffee farmer in Aceh. The game blends interactive storytelling with realistic farming simulations to educate players on the entire coffee farming process.

Year: 2025
Role: Game Developer
Course: Educational and Simulation Games
Repository: https://github.com/Aliyah-Pattoza/FP-GAYOCRAFT.git

Project Description

GAYOCRAFT is a visual novel-style simulation game that places players in the role of a young aspiring Gayo coffee farmer in Aceh. Throughout the game, players are methodically guided through the coffee farming process, encompassing land preparation, seed selection, planting, crop maintenance, harvesting, and post-harvest handling.

Key Features

  • Educational Simulation: Detailed instructions and decision-making processes that reflect actual Gayo coffee farming techniques.
  • Interactive Storyline: Players make choices that affect their progress and outcomes, simulating challenges and rewards of real farming.
  • Economic System: After harvesting, players can sell their coffee yields, manage income, and reinvest to expand their farm.
  • Achievement-based Ending: The story culminates with the player becoming the most successful and well-known Gayo coffee plantation owner.

Study Buddy – Academic Messaging Platform

Developed Study Buddy, a mobile application designed as an Academic-First Messaging Platform that integrates academic communication, task management, and collaborative learning into one unified ecosystem. This project was built to address the real-world challenges faced by Indonesian students in managing their academic life more effectively and separately from personal messaging platforms like WhatsApp.

Year: 2025
Role: UI/UX
Course: Software Design

Prototype: Prototype-Study-Buddy

Project Description

Study Buddy is a comprehensive digital solution tailored for students to enhance academic productivity. It reimagines how students communicate, organize, and collaborate in the academic context by offering key features such as:

SOS - Multiplayer Game

SOS Game is a two-player network-based digital board game built with Python and socket programming, allowing players to play in real-time in one room with a scoring system based on the formation of the word “S-O-S”.

Year: 2025
Role: Fullstack Developer
Course: Network Programming
Repository: https://github.com/farrasblah/sos_game

Project Description

SOS Game is a networked digital board game designed to enable two players to compete in real time over the internet within a shared game room. The game's construction utilized Python with Pygame for the graphical user interface, enabling players to engage in a strategic sequence of actions. In this sequence, players alternately place the letter "S" or "O" on a grid of selectable sizes (3x3, 5x5, or 9x9). The objective of the game is to form the sequence "S-O-S" in any of the following orientations: horizontally, vertically, or diagonally. The player who achieves this formation first earns points. The game concludes when the board is full, and the winner is determined by the highest score—or a draw if scores are tied. The game employs a client-server architecture with a manually implemented HTTP-like protocol utilizing Python socket programming. The server concurrently manages multiple connections via threading, overseeing all active game states (player list, turn order, board status, and scores) in memory based on a unique room_id.

Pawfee – Cafe Website

Created a cafe web application called Pawfee, developed using Golang, which enables users to place food and drink orders, make table reservations, and manage their interactions with the cafe entirely online. The project was developed for the Framework-based Programming course and focuses on simulating a modern digital ordering system with real-time features and admin control panels.

Year: 2024
Role: Full-stack Web Developer
Course: Framework-based Programming
Repository: https://github.com/Aliyah-Pattoza/PBKK-FP-GO.git

Project Description

Pawfee is a web application that is based on the Go programming language. It allows cafe customers to access a menu that is organized into categories, place orders for food and beverages, and reserve tables—all from their browsers. The system implements a clear separation between customer and administrative roles. It is possible for customers to engage with a dynamic menu, view the contents of their cart, and ascertain their reservation status. Administrators are able to manage orders, menu items, and reservations through a secure backend.

Key Features

  • Secure Authentication: Role-based login system for customers and admins, with session handling.
  • Dynamic Menu Interaction: Customers can increment or decrement item quantities, with real-time cart updates and availability checks.
  • Self-Ordering System: Seamless digital ordering workflow with item selection, cart preview, and order confirmation.
  • Table Reservation: Simple reservation form with customer phone input and reservation status tracking.
  • Admin Panel: Tools to view/edit menu items, manage order statuses, confirm/cancel reservations, and view user activity.

PETET – Petualangan Teman TK

PETET is a Java-based 2D educational game that invites the player, a kindergartner, on an adventure to collect school supplies while avoiding obstacles to reach the school with the best score.

Year: 2023
Role: Game Developer
Course: Object-Oriented Programming
Repository: https://github.com/Aliyah-Pattoza/PBO-FP.git

Project Description

The PETET game is a two-dimensional educational adventure game developed using Java for the Object Oriented Programming course. The game in question simulates a kindergartner's journey to school by collecting various school supplies scattered along the way. Each item contributes to the player's score. The objective of the game is to navigate a grid-based map while evading obstacles such as mud and water puddles, which result in a points deduction when stepped on. The objective of the game is to reach the school building, which serves as the culmination of the game and displays the total score based on the items collected. The game places emphasis on object-oriented principles, including inheritance, encapsulation, and polymorphism, in the management of game entities such as items, obstacles, characters, and scoring logic.

Java 2D Game OOP Game Development
